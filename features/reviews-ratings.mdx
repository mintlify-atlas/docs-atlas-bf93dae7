The reviews and ratings system allows customers to share their experiences with products, helping other shoppers make informed decisions. The system supports both authenticated and anonymous reviews with a 5-star rating scale.

## Overview

Customers can leave reviews on any product, rating it from 1 to 5 stars and adding optional text comments. Reviews are displayed on product detail pages, showing the reviewer's name, rating, comment, and date.

<Card title="Review System Features" icon="star">

  - Optional text comments

  - Anonymous reviews for non-logged-in users
  - Timestamp for each review
  - Average rating calculation
  - Review count display
  - Real-time review submission
  - Validation of rating and comment
</Card>

## Database Schema

### Reviews Table (Valoraciones)

```sql
CREATE TABLE IF NOT EXISTS valoraciones (
  id INT AUTO_INCREMENT PRIMARY KEY,
  producto_id INT NOT NULL,
  usuario_id INT,
  calificacion TINYINT NOT NULL CHECK (calificacion BETWEEN 1 AND 5),
  comentario TEXT,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (producto_id) REFERENCES productos(id) ON DELETE CASCADE,
  FOREIGN KEY (usuario_id) REFERENCES usuarios(id) ON DELETE SET NULL
);

CREATE INDEX idx_valoraciones_producto ON valoraciones(producto_id);
```

<Note>
  The `usuario_id` can be NULL to support anonymous reviews. When a user is deleted, their reviews remain but show as anonymous.
</Note>

### Schema Details

| Field | Type | Description |
|-------|------|-------------|
| `id` | INT | Unique review identifier |
| `producto_id` | INT | Product being reviewed (required) |
| `usuario_id` | INT | User who wrote review (nullable) |
| `calificacion` | TINYINT | Star rating 1-5 (required) |
| `comentario` | TEXT | Review text (optional) |
| `created_at` | TIMESTAMP | When review was created |

## User Features

### Viewing Reviews

Reviews are displayed on the product detail page in a dedicated tab:

```jsx
// From ProductDetail.jsx
const [reviews, setReviews] = useState([]);
const [loadingReviews, setLoadingReviews] = useState(false);

useEffect(() => {
  const fetchProductAndReviews = async () => {
    // Fetch product data
    const productResponse = await api.get(`/productos/${productId}`);
    setProduct(productResponse.data);
    
    // Fetch reviews
    try {
      setLoadingReviews(true);
      const reviewsResponse = await api.get(`/reviews?producto_id=${productId}`);
      const reviewsData = Array.isArray(reviewsResponse.data)
        ? reviewsResponse.data
        : [];
      setReviews(reviewsData);
    } catch (reviewError) {
      console.error('Error cargando reseñas:', reviewError);
      setReviews([]);
    } finally {
      setLoadingReviews(false);
    }
  };
  
  fetchProductAndReviews();
}, [productId]);
```

### Average Rating

The average rating is calculated and displayed at the top of the product:

```jsx
// Calculate average rating
const avgRating = reviews.length > 0
  ? reviews.reduce((sum, r) => sum + (r.calificacion || 0), 0) / reviews.length
  : 0;

// Display average rating
<div className="flex items-center gap-4">
  <div className="flex items-center gap-2">
    <div className="flex">
      {[1, 2, 3, 4, 5].map((star) => (
        <Star
          key={star}
          className={`w-5 h-5 fill-current ${
            star <= Math.round(avgRating) 
              ? 'text-yellow-400' 
              : 'text-muted/30'

        />
      ))}

      <span className="text-sm text-muted">

      </span>
    ) : (
      <span className="text-sm text-muted">Sin reseñas</span>
    )}
  </div>
</div>
```

### Writing a Review

Logged-in users can submit reviews through a form:

```jsx
const [reviewText, setReviewText] = useState("");
const [rating, setRating] = useState(0);
const [isSubmitting, setIsSubmitting] = useState(false);
const [submitError, setSubmitError] = useState(null);

const handleReviewSubmit = async () => {
  // Validate
  if (rating === 0 || !reviewText.trim()) {
    setSubmitError("Por favor, completa todos los campos.");
    return;
  }
  
  setIsSubmitting(true);
  setSubmitError(null);
  
  try {
    // Submit review
    await api.post("/reviews", {
      producto_id: productId,
      usuario_id: user?.id || "unknown",
      calificacion: rating,
      comentario: reviewText.trim(),
    });
    
    // Reload reviews
    const updatedReviews = await api.get(`/reviews?producto_id=${productId}`);
    setReviews(Array.isArray(updatedReviews.data) ? updatedReviews.data : []);
    
    // Reset form
    setReviewText("");
    setRating(0);
    alert("¡Gracias por tu reseña!");
  } catch (error) {
    console.error('Error al enviar reseña:', error);
    setSubmitError(
      error.message || "Hubo un problema al enviar tu reseña."
    );
  } finally {
    setIsSubmitting(false);
  }
};
```

          value={reviewText}

        />
        {reviewText.trim().length === 0 && (
          <p className="text-sm text-rose-400">Por favor, escribe tu reseña</p>
        )}
      </div>
      
      {/* Error message */}
      {submitError && (
        <div className="p-4 bg-rose-500/10 border border-rose-500/30 text-rose-400 rounded-xl text-sm">
          {submitError}
        </div>
      )}
      
      {/* Submit button */}

        type="submit"
        disabled={isSubmitting || rating === 0 || reviewText.trim().length === 0}
        className="btn-add-cart !px-8 !py-3"
      >
        {isSubmitting ? (

            Enviando...
          </>
        ) : (

            <User className="w-5 h-5" />

          </>
        )}
      </button>
    </form>

## Backend API

### Get Reviews for Product

```javascript
// GET /api/reviews/producto/:productId
router.get('/producto/:productId', async (req, res) => {
  try {
    const { productId } = req.params;
    const result = await db.query(
      `SELECT v.id, v.calificacion, v.comentario, v.created_at,
         u.nombre as usuario_nombre, u.email as usuario_email
       FROM valoraciones v
       LEFT JOIN usuarios u ON v.usuario_id = u.id
       WHERE v.producto_id = $1
       ORDER BY v.created_at DESC`,
      [productId]
    );
    res.json(result.rows);
  } catch (error) {
    console.error('Error al obtener reseñas:', error);
    res.status(500).json({ error: 'Error al cargar reseñas' });
  }
});
```

<Note>
  Reviews are joined with the `usuarios` table to get the reviewer's name, but `usuario_id` can be NULL for anonymous reviews.
</Note>

### Create Review

```javascript
// POST /api/reviews
router.post('/', async (req, res) => {
  try {
    const { producto_id, usuario_id, calificacion, comentario } = req.body;
    
    // Validate
    if (!producto_id || !calificacion || calificacion < 1 || calificacion > 5) {
      return res.status(400).json({ error: 'Datos inválidos' });
    }
    
    // Insert review
    const result = await db.query(
      `INSERT INTO valoraciones (producto_id, usuario_id, calificacion, comentario)
       VALUES ($1, $2, $3, $4)
       RETURNING id`,
      [producto_id, usuario_id || null, calificacion, comentario || null]
    );
    
    // Update product averages (optional)
    await db.query(
      `UPDATE productos p
       SET 
         total_valoraciones = (SELECT COUNT(*) FROM valoraciones v WHERE v.producto_id = p.id),
         promedio_calificacion = (SELECT AVG(calificacion) FROM valoraciones v WHERE v.producto_id = p.id)
       WHERE p.id = $1`,
      [producto_id]
    );
    
    res.status(201).json({
      id: result.rows[0].id,
      producto_id,
      calificacion,
      comentario
    });
  } catch (error) {
    console.error('Error al crear reseña:', error);
    res.status(500).json({ error: 'Error al guardar la reseña' });
  }
});
```

### Validation Rules

<Card title="Review Validation" icon="check">
  - **Rating**: Required, must be integer between 1-5
  - **Product ID**: Required, must reference existing product
  - **User ID**: Optional, can be NULL for anonymous reviews
  - **Comment**: Optional text field, no length limit
  - **All reviews**: Automatically timestamped with `created_at`
</Card>

## Authentication Flow

### For Logged-In Users

```jsx
function useAuth() {
  const [user, setUser] = useState(null);
  const [loading, setLoading] = useState(true);
  
  useEffect(() => {
    const token = localStorage.getItem("token");
    if (token) {
      const userData = localStorage.getItem("user");
      setUser(userData ? JSON.parse(userData) : { id: "unknown" });
    }
    setLoading(false);
  }, []);
  
  return { user, loading, isLoggedIn: !!localStorage.getItem("token") };
}

const { user, loading: authLoading, isLoggedIn } = useAuth();
```

## Best Practices

<Card title="Implementation Tips" icon="lightbulb">
  1. **Always validate ratings** - Ensure ratings are 1-5
  2. **Handle NULL users** - Support both logged-in and anonymous reviews
  3. **Sanitize comments** - Prevent XSS attacks in review text
  4. **Index producto_id** - Improve query performance
  5. **Show newest first** - Order by `created_at DESC`
  6. **Graceful degradation** - Handle missing reviews gracefully
  7. **Calculate on-demand** - Compute averages in real-time
  8. **Allow editing** - Consider adding edit/delete for review authors
</Card>
