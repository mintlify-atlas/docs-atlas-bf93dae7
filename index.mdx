## Introduction

Black Michi Studio is a modern e-commerce platform built with React and Express, designed to deliver a seamless shopping experience for 3D products. The platform features integrated payment processing through Flow, comprehensive order management, and performance optimizations for fast loading times.

## Technology Stack

Built with modern technologies for performance and scalability.

- **Frontend**: React 18, Vite, Tailwind CSS, React Router
- **Backend**: Node.js, Express, PostgreSQL
- **Authentication**: JWT, Google OAuth, bcrypt
- **Payments**: Flow payment gateway integration
- **Storage**: Supabase for media assets
- **Deployment**: Docker, Render-ready configuration
