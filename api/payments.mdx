## Create Payment

<CodeGroup>
```bash cURL
curl -X POST https://api.blackmichistudio.com/api/payments/flow/create \
  -H "Content-Type: application/json" \
  -d '{
    "items": [
      {
        "id": 123,
        "nombre": "Product Name",
        "precio": 15990,
        "quantity": 2,
        "imagen": "https://example.com/image.jpg"
      }
    ],
    "total": 35480,
    "email": "customer@example.com",
    "nombre": "Juan Pérez",
    "telefono": "+56912345678",
    "direccion": "Av. Libertador 1234, Santiago",
    "notas": "Dejar en portería"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.blackmichistudio.com/api/payments/flow/create', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    items: [
      {
        id: 123,
        nombre: 'Product Name',
        precio: 15990,
        quantity: 2,
        imagen: 'https://example.com/image.jpg'
      }
    ],
    total: 35480, // items total (31990) + shipping (3500)
    email: 'customer@example.com',
    nombre: 'Juan Pérez',
    telefono: '+56912345678',
    direccion: 'Av. Libertador 1234, Santiago',
    notas: 'Dejar en portería'
  })
});

const result = await response.json();
if (result.success) {
  window.location.href = result.paymentUrl;
}
```

```python Python

response = requests.post(
  'https://api.blackmichistudio.com/api/payments/flow/create',

    'nombre': 'Juan Pérez',
    'telefono': '+56912345678',
    'direccion': 'Av. Libertador 1234, Santiago',
    'notas': 'Dejar en portería'
  }
)

result = response.json()
```

## Payment Confirmation Webhook

<Warning>
  This endpoint is called by Flow servers only. Do not call it directly from your application.
</Warning>

### Endpoint

<ParamField path="method" type="string">
  POST
</ParamField>

<ParamField path="endpoint" type="string">
  /api/payments/flow/confirmation
</ParamField>

<ParamField path="authentication" type="string">
  Flow signature validation (production only)
</ParamField>

### Request Body

<ParamField body="token" type="string" required>
  Flow payment token
</ParamField>

### Process Flow

1. **Webhook Logging**: Logs all incoming webhooks to `flow_webhooks` table
2. **Signature Validation**: Validates Flow signature (production only)
3. **Payment Status**: Fetches payment status from Flow API
4. **Order Update**: Updates order status based on Flow payment status
5. **Stock Deduction**: If payment is successful (status = 2), deducts stock from products

### Flow Status Mapping

| Flow Status | Internal Status | Description |
|------------|-----------------|-------------|
| 1 | pendiente | Payment pending |
| 2 | pagado | Payment successful |
| 3 | rechazado | Payment rejected |
| 4 | cancelado | Payment cancelled |

### Response

- **200**: Webhook processed successfully
- **400**: Invalid signature or missing token
- **404**: Order not found
- **500**: Processing error

### Automatic Operations on Success

- **Stock Deduction**: Reduces product stock by ordered quantities
- **Payment Date**: Sets `fecha_pago` to current timestamp
- **Flow Data**: Stores complete Flow payment data in `flow_payment_data` field

## Get Order Status

<CodeGroup>
```bash cURL
curl https://api.blackmichistudio.com/api/payments/pedido/123/status
```

```javascript JavaScript
const pedidoId = 123;
const response = await fetch(
  `https://api.blackmichistudio.com/api/payments/pedido/${pedidoId}/status`
);
const data = await response.json();
```

```python Python
import requests

pedido_id = 123
response = requests.get(
  f'https://api.blackmichistudio.com/api/payments/pedido/{pedido_id}/status'
```
