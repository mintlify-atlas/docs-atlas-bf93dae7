## Get Product Reviews

<CodeGroup>
```bash cURL
curl https://api.blackmichistudio.com/api/reviews/producto/123
```

```javascript JavaScript
const productId = 123;
const response = await fetch(
  `https://api.blackmichistudio.com/api/reviews/producto/${productId}`
);
const reviews = await response.json();
```

```python Python
import requests

product_id = 123
response = requests.get(
  f'https://api.blackmichistudio.com/api/reviews/producto/{product_id}'
)
reviews = response.json()
```
</CodeGroup>

## Create Review

<CodeGroup>
```bash cURL
curl -X POST https://api.blackmichistudio.com/api/reviews \
  -H "Content-Type: application/json" \
  -d '{
    "producto_id": 123,
    "usuario_id": 456,
    "calificacion": 5,
    "comentario": "Excelente producto, muy recomendado!"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.blackmichistudio.com/api/reviews', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    producto_id: 123,
    usuario_id: 456, // optional
    calificacion: 5,
    comentario: 'Excelente producto, muy recomendado!'
  })
});

const review = await response.json();
```

```python Python
import requests

response = requests.post(
  'https://api.blackmichistudio.com/api/reviews',
  json={
    'producto_id': 123,
    'usuario_id': 456,
    'calificacion': 5,
    'comentario': 'Excelente producto, muy recomendado!'
  }
)

review = response.json()
```
</CodeGroup>

### Automatic Operations

- **Product Update**: Automatically recalculates and updates the product's:
  - `total_valoraciones`: Total count of reviews
  - `promedio_calificacion`: Average rating across all reviews

### Validation Rules

- `producto_id` is required
- `calificacion` must be between 1 and 5 (inclusive)
- `comentario` is optional
- `usuario_id` is optional (allows guest reviews)

## Rating System

### Rating Scale

- **1 star**: Poor
- **2 stars**: Fair
- **3 stars**: Good
- **4 stars**: Very Good
- **5 stars**: Excellent

### Anonymous Reviews

This API allows anonymous reviews by setting `usuario_id` to `null`. This is useful for guest checkout scenarios where users can leave reviews without creating an account.

### Product Rating Updates

When a review is created, the product's rating statistics are automatically updated:

```sql
UPDATE productos p
SET 
  total_valoraciones = (SELECT COUNT(*) FROM valoraciones v WHERE v.producto_id = p.id),
  promedio_calificacion = (SELECT AVG(calificacion) FROM valoraciones v WHERE v.producto_id = p.id)
WHERE p.id = $1
```

This ensures that product listings always show current rating information without requiring additional queries.
