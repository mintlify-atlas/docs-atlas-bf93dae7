## Register User

<CodeGroup>
```bash cURL
curl -X POST https://api.blackmichistudio.com/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "nombre": "John Doe",
    "email": "john@example.com",
    "password": "securePassword123"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.blackmichistudio.com/api/auth/register', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    nombre: 'John Doe',
    email: 'john@example.com',
    password: 'securePassword123'
  })
});

const data = await response.json();
```

```python Python
import requests

response = requests.post(
  'https://api.blackmichistudio.com/api/auth/register',
  json={
    'nombre': 'John Doe',
    'email': 'john@example.com',
    'password': 'securePassword123'
  }
)

data = response.json()
```
</CodeGroup>

### Endpoint

<ParamField path="method" type="string">
  POST
</ParamField>

<ParamField path="endpoint" type="string">
  /api/auth/register
</ParamField>

<ParamField path="authentication" type="string">
  None required
</ParamField>

### Request Body

<ParamField body="nombre" type="string" required>
  User's full name
</ParamField>

<ParamField body="email" type="string" required>
  User's email address (must be unique)
</ParamField>

<ParamField body="password" type="string" required>
  User's password (will be hashed with bcrypt)
</ParamField>

### Response

<ResponseField name="user" type="object">
  User information
  <expandable>
    <ResponseField name="id" type="integer">
      User's unique identifier
    </ResponseField>
    <ResponseField name="nombre" type="string">
      User's full name
    </ResponseField>
    <ResponseField name="email" type="string">
      User's email address
    </ResponseField>
    <ResponseField name="rol" type="string">
      User role (default: "cliente")
    </ResponseField>
  </expandable>
</ResponseField>

<ResponseField name="token" type="string">
  JWT authentication token (expires in 7 days)
</ResponseField>

### Error Responses

<ResponseField name="400" type="object">
  Bad Request - Registration failed
  <expandable>
    <ResponseField name="error" type="string">
      "Error al registrar usuario" - Email may already exist or invalid data
    </ResponseField>
  </expandable>
</ResponseField>

## Google OAuth Login

<CodeGroup>
```bash cURL

  -H "Content-Type: application/json" \
  -d '{
    "idToken": "eyJhbGciOiJSUzI1NiIsImtpZCI6..."
  }'
```

```javascript JavaScript
const response = await fetch('https://api.blackmichistudio.com/api/auth/google-login', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },

    idToken: googleIdToken // from Google Sign-In
```

## JWT Token Format

All successful authentication responses return a JWT token that contains:

```json
{
  "id": 123,
  "email": "john@example.com",
  "rol": "cliente",
  "iat": 1234567890,
  "exp": 1234567890
}
```

### Using the Token

Include the token in the `Authorization` header for protected endpoints:

```bash
Authorization: Bearer <your-jwt-token>
```
