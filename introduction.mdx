# Welcome to Black Michi Studio

Black Michi Studio is a modern, full-stack e-commerce platform built specifically for selling 3D printed products, custom merchandise, and collectibles. The platform features a React-based frontend with a robust Node.js/Express backend, PostgreSQL database, and integrated payment processing through Flow.

## What is Black Michi Studio?

Black Michi Studio is a complete e-commerce solution that enables you to:

- **Sell 3D Printed Products**: Showcase and sell custom 3D printed items like personalized glasses, decorative plaques, and collectible figures
- **Manage Inventory**: Track product stock, categories, and pricing with real-time updates
- **Process Payments**: Integrate with Flow payment gateway for secure Chilean payment processing
- **Handle Orders**: Complete order management from cart to delivery tracking
- **Admin Dashboard**: Full-featured admin panel for managing products, orders, and customer data

## Key Features

### Frontend Capabilities

- **Modern React UI**: Built with React 18, Vite, and Tailwind CSS
- **Responsive Design**: Mobile-first design that works on all devices
- **Product Browsing**: Category-based navigation, search, and filtering
- **Shopping Cart**: Persistent cart with real-time calculations
- **User Authentication**: Secure login and registration with JWT tokens
- **Payment Integration**: Seamless Flow payment gateway integration
- **Order Tracking**: Real-time order status updates
- **Product Reviews**: Customer ratings and review system

### Backend Capabilities

- **RESTful API**: Clean API architecture with Express.js
- **PostgreSQL Database**: Robust relational database with optimized queries
- **Image Processing**: Automatic image optimization with Sharp
- **File Upload**: Multer-based file upload for product images
- **Authentication**: JWT-based authentication with bcrypt password hashing
- **Payment Processing**: Flow payment gateway integration with webhook handling
- **Stock Management**: Automatic stock deduction on successful payments
- **Admin Controls**: Role-based access control for admin operations

## Database Schema Highlights

The platform uses a comprehensive PostgreSQL schema with the following core tables:

- **usuarios**: User accounts with role-based access (admin/cliente)
- **categorias**: Product categories with custom styling
- **productos**: Product catalog with pricing, stock, and images
- **pedidos**: Orders with Flow payment integration
- **pedido_items**: Order line items preserving product details at purchase time
- **flow_webhooks**: Payment webhook logging for audit trails
- **valoraciones**: Product reviews and ratings
- **featured_productos**: Featured product management
- **hero_images**: Homepage hero image configuration
